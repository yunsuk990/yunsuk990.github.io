AI를 처음 공부할 때 가장 헷갈리는 건 용어입니다.
이 글은 **AI 입문자가 방향을 잃지 않도록**, 꼭 필요한 개념만 먼저 정리한 문서입니다.

---

## 1) AI, 머신러닝, 딥러닝은 어떻게 다를까?

### AI (Artificial Intelligence)
사람의 지능적인 행동(예: 인식, 판단, 추천)을 컴퓨터가 하도록 만드는 전체 분야입니다.

### 머신러닝 (Machine Learning)
AI를 만드는 방법 중 하나입니다.
규칙을 사람이 직접 다 짜는 대신, **데이터로부터 패턴을 학습**합니다.

### 딥러닝 (Deep Learning)
머신러닝의 하위 분야입니다.
신경망(Neural Network)을 깊게 쌓아 이미지, 음성, 텍스트 같은 복잡한 데이터를 잘 다룹니다.

즉, 관계는 아래처럼 생각하면 쉽습니다.

- **AI ⊃ 머신러닝 ⊃ 딥러닝**

---

## 2) AI가 실제로 하는 일

대표적인 문제 유형은 아래 4가지입니다.

1. **분류(Classification)**
   - 예: 스팸 메일 / 정상 메일 구분
2. **회귀(Regression)**
   - 예: 집값 예측, 매출 예측
3. **군집화(Clustering)**
   - 예: 비슷한 사용자 그룹 자동 분류
4. **생성(Generation)**
   - 예: 텍스트 생성, 이미지 생성

---

## 3) AI 학습의 기본 흐름

AI 프로젝트는 보통 이 순서로 진행됩니다.

1. 문제 정의 (무엇을 예측/분류할지)
2. 데이터 수집
3. 데이터 전처리 (결측치, 이상치, 정규화 등)
4. 모델 선택
5. 학습(Train) / 검증(Validation)
6. 평가(Test)
7. 배포 및 모니터링

핵심은 **모델보다 데이터 품질**입니다.
좋은 데이터가 성능의 대부분을 결정합니다.

---

## 4) 꼭 알아야 할 용어 8개

- **Feature(특징)**: 입력 변수
- **Label(정답)**: 모델이 맞춰야 하는 값
- **Train/Validation/Test**: 학습/튜닝/최종평가 데이터 분리
- **Overfitting(과적합)**: 학습 데이터만 잘 맞고 일반화 실패
- **Underfitting(과소적합)**: 학습 자체가 부족한 상태
- **Loss(손실함수)**: 모델의 오차를 수치화한 값
- **Epoch**: 전체 데이터를 한 번 학습한 횟수
- **Inference**: 학습된 모델로 실제 예측하는 단계

---

## 5) 비전공자/초보자 기준 공부 순서 (추천)

### 1단계: 기초 체력
- Python 기본 문법
- 선형대수(벡터, 행렬) 기초
- 확률/통계 기초(평균, 분산, 정규분포)

### 2단계: 머신러닝 기초
- 지도학습 / 비지도학습
- 회귀, 분류 알고리즘
- 평가 지표(Accuracy, Precision, Recall, F1)

### 3단계: 딥러닝 기초
- 퍼셉트론, 신경망, 활성화 함수
- 역전파(Backpropagation)
- CNN/RNN/Transformer 개념

### 4단계: 실전
- 작은 데이터셋으로 미니 프로젝트
- 성능 개선 실험
- 결과 정리 및 회고
