Flutter를 시작하기 전에 알아두면 좋은 **Dart 기본 문법**을 짧게 정리합니다.

## 1) 변수 선언과 타입

```dart
var name = 'Yunsuk';      // 타입 추론
String title = 'Developer';
final createdAt = DateTime.now(); // 한 번만 할당
const pi = 3.14;          // 컴파일 타임 상수
```

- `var`: 타입 추론
- `final`: 런타임 상수
- `const`: 컴파일 타임 상수

## 2) 함수

```dart
int add(int a, int b) {
  return a + b;
}

String greet(String name, {String prefix = 'Hi'}) {
  return '$prefix, $name';
}
```

- named parameter를 사용하면 가독성이 좋아집니다.
- 기본값을 지정해 두면 호출 코드가 단순해집니다.

## 3) Null Safety

```dart
String? nickname;          // null 허용
String username = 'ys990';

print(nickname?.length);   // null-safe 호출
print(username.length);
```

- `?`: null 허용 타입
- `?.`: null-safe 접근
- `!`: null 아님을 보장(필요할 때만 사용)

## 4) 컬렉션(List, Map)

```dart
final numbers = [1, 2, 3];
final profile = {'name': 'Yunsuk', 'job': 'Android Dev'};

numbers.add(4);
print(profile['name']);
```

실무에서는 List/Map을 JSON 데이터 처리에 자주 사용합니다.

## 정리

Dart는 문법이 간결하고 null-safety가 강력해서 Flutter 개발 생산성을 높여줍니다.
다음 글에서는 클래스와 비동기(`Future`, `async/await`)를 정리해보겠습니다.
